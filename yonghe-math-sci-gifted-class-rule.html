《新北市立永和國民中學數理資優班自治條例》

簡稱《班規》

圖例：新增　刪除

1114修正案2024/11/23 07:00公告；2024/11/25 00:00起實施

  1. 總論
    1. 公告及實施規定如下：
      1. 若於週一至週五之間公告，於隔週一實施。
      2. 若於週六至週日之間公告，於隔週之週一實施。
  2. 獎勵＆處罰
    1. 獎勵及處罰點數以違反該條，以該條之點數增加為原則。
    2. 獎勵辦法如下：
      1. 班會課口頭報告（３分鐘／記獎勵１００點），須包含簡報內容；若無簡報，獎勵點數由導師評定（獎勵點數＜１０）。
      2. 理化或數學課回答問題（１個問題／記獎勵１～５點，不得超過）次數由上課老師評定。
      3. 寫每週研究進度心得（１００字／記獎勵１～５點）心得內容由導師評定。
      4. 愛班服務（清板溝、掃＋拖……）記獎勵０～５０點。
      5. 學期末獎勵與處罰點數互抵，若互抵後尚有處罰點數未抵完，抄寫元素週期表１～１１８號元素（折抵處罰１０～１５點）。
    3. 抵銷限制：處罰點低於１０點只能使用上課回答問題抵銷。
    4. 每２００獎勵點可選抵消４次藍本ｏｒ一支嘉獎。
    5. 每１２００處罰點記一支警告。
  3. 獨立研究＆Ｎｏｔｉｏｎ檢核
    1. 幹部檢核後以Ｇｏｏｇｌｅ試算表紀錄，若有組別進度太落後，由栩、濬、毅、亦做關心。
    2. Ｎｏｔｉｏｎ的檢核標準由幹部產出，置於此。
    3. 檢核當日未完成扣１０點開始，ｄｅｌａｙ　１天多扣２點（段考周圍的幾週會放寬一點，例如，每兩天結算一次。）
    4. Ｎｏｔｉｏｎ複核的扣點以表單完成時間為準。
  4. 課堂＆會議規定
    1. 班會或其他會議若無故缺席，記處罰２點；若無故遲到，記處罰１點。
    2. 第八節課程若１６：１５前準時到，記獎勵１點；若超過１６：２０無證明未到，記處罰１點。
    3. 未完整做值日生之工作（包括擦黑板、關冷氣、關電扇、關燈、檢查有無垃圾、排桌椅……等）者，記處罰１點；若累犯，則每次遞加２點。
    4. 上課時若太吵，經老師確認後記處罰１～５點不等。
    5. 不得丟擲任何物品（如：粉筆、球類……等），違者記處罰１點。
    6. 資三教室走廊的黑板可畫圖、塗鴉，但若有不雅或侮辱性相關圖案、文字，記處罰１０點。
      不得資三教室走廊的黑板上隨意塗鴉，影響版面、遮住重要內容，違者記處罰２點。
    7. 若比中指、說帶有性別歧視、黃色言論被舉報，記處罰１～５點不等。
    8. 準時完成課堂任務，記獎勵５～１０點。
  5. 作業相關規定
    1. 選修課程期限前交不出作業扣３點，ｄｅｌａｙ　１天扣１～２點，如果在下一次選修課前都還沒交出，再加扣１０點。
    2. 藍本準時當週每天的１０：００前繳交並簽名，並且當天第八節拿回，可加獎勵１０點。
    3. 從１０／１４開始，未寫扣點第一次３點、第二次５點，接下來以費氏數列計算下去。（例如：８、１３、２１、３４……）
  6. 隱藏任務
    1. 每次有新公告，皆可能會發布新隱藏任務。
    2. 若未於期限內完成隱藏任務，將記處罰１～１０點。
    3. 若有題目為答錯，記處罰１點。
    4. 若於某日期前（於各次隱任內指定）先完成隱藏任務，記獎勵５點。
